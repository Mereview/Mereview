# 요구사항 명세


## ✔회원

| 중요도 | 기능 | 요구사항명 | 요구사항 내용 | 비고 |
| --- | --- | --- | --- | --- |
| 상 | 가입 | 사용자 등록 | 이메일, 비밀번호, 닉네임, 성별, 생일, 관심사, 프로필 이미지 등을 입력받는다 | 1 . 이메일 작성시 도메인은 드롭다운으로 선택하도록 한다. |
| 하 |  | 이메일 인증 | 작성된 이메일로 인증 메일을 보낸다 |  |
| 하 |  | 중복 검사 | 이미 등록된 이메일인지 중복 검사를 한다 |  |
| 하 |  | 중복 검사 | 이미 등록된 닉네임인지 중복 검사를 한다 |  |
| 상 |  | 유효성 검사 | 비밀번호가 특수문자, 숫자, 영어를 포함하고 8~20글자 사이인지 유효성 검사를 한다 | 2. 비밀번호는 숫자, 영어 8자 이상,  특수문자 1개 이상 반드시 포함 |
| 상 | 로그인 및 인증 | 조건 | 이메일과 비밀번호를 입력하여 로그인 할 수 있다 | 이메일은 xxx@xxx.xxx 형식 비밀번호는 8자 이상,  특수문자(!@#$%^&*) 1개 이상 포함 |
| 중 |  | 로그인 인증 | 페이지에 접근 권한이 없을 시 로그인 화면으로 이동시킨다 |  |
| 하 |  | 보안 | 일정 시간이 지나면 세션이 만료되어 다시 로그인 시킨다 | 로그인 지속시간 - 24시간 |
| 상 | 조회 | 프로필 | 해당 회원의 정보(회원 정보, 업적, 관심사, 티어)를 불러올 수 있다 | 마이페이지 |
| 중 |  | 리뷰 조회 | 회원이 작성한 리뷰, 댓글, 평가를 남긴 리뷰 등을 확인 가능하다. |  |
| 중 |  | 방문자수 | 전체 방문자, 일일 방문자 수 조회가 가능하다. |  |
| 상 | 티어 | 티어 | 장르별 티어가 존재한다. 산정방식 |  |
|  |  | 업적 | 장르별 업적이 존재한다. 산정방식 |  |
| 중 | 수정 | 정보 수정 | 프로필 이미지, 비밀번호, 관심사를 변경할 수 있다 |  |
| 중 | 팔로우 | 등록 | 다른 사용자를 팔로우 할 수 있다 |  |
|  |  | 취소 | 다른 사용자에 대한 팔로우 취소가 가능하다. |  |
| 중 | 탈퇴 | 회원 탈퇴 | 회원 탈퇴를 진행한다. | 회원 탈퇴 시 삭제하지 않고 회원 권한을 탈퇴회원으로 변경한다. |

## ✔리뷰

| 중요도 | 기능 | 요구사항명 | 요청사항 내용 | 비고 |
| --- | --- | --- | --- | --- |
| 상 | 등록 | 리뷰 등록 | 리뷰제목, 영화제목, 배경이미지, 한줄평, 리뷰내용(사진포함가능), 영화에 대한 평가, 키워드를 등록할 수 있다 | 작성 시 해당 장르를 관심사로 등록한 회원에게 알림이 간다. 영화 제목 자동완성 배경이미지 첨부하지 않을 경우 포스터로(포스터도 없는 경우 흰색 배경으로) 설정
키워드 추가 |
| 중 |  | 키워드 추천 | 키워드  작성중 실시간으로 추천 항목을 받아볼 수 있음 | 비동기 |
| 중 |  | 댓글 등록 | 리뷰 밑에 댓글을 달 수 있다 |  |
| 상 | 조회 | 리뷰 조회 | 리뷰 상세 및 목록 조회가 가능하다. | 관심사 / 장르별 조회, 팔로우 한 회원 조회, 검색 기능 제공 목록조회 시 정렬 기준 선택 가능 - 최신순, 추천순(기간선택), 조회순, 랜덤 정렬 |
| 중 | 수정 | 리뷰 수정 | 리뷰 작성자는 내용을 수정 할 수 있다 |  |
| 중 |  | 댓글 수정 | 댓글 작성자는 내용을 수정 할 수 있다 |  |
| 중 | 삭제 | 리뷰 삭제 | 리뷰 작성자는 내용을 삭제 할 수 있다 | CASCADE |
| 중 |  | 댓글 삭제 | 댓글 작성자는 내용을 삭제 할 수 있다 | CASCADE |
| 상 | 평가 | 영화 평가 | 리뷰 작성자의 평가를 통해 영화의 평가별 개수를 산출할 수 있다 | count |
| 상 |  | 리뷰 평가 | 사용자들은 리뷰에 재밌어요 / 유용해요 / 별로에요를 남길 수 있고 이를 통해 리뷰 작성자의 티어가 결정된다 | 재밌어요, 유용해요 : + 5점 씩 별로에요 : -5 점 재미, 유용은 서로 중복이 가능하나, 별로에요는 다른 항목과 선택될 수 없음 |
| 중 |  | 댓글 평가 | 사용자들은 리뷰속 댓글에 좋아요 / 싫어요 이모티콘을 남길 수 있다 |  |
| 중 | 알림 | 랜덤 알림 | 리뷰 작성 완료시 해당 장르를 관심등록한 랜덤한 사용자에게 알림이 갈 수 있다 | SQL 랜덤 정렬, NavBar에 알림 아이콘 |
