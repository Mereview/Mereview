# API 명세서

## 👩 회원

| 이름 | 내용 | 입력 정보 | 출력 정보 | url | method |
| --- | --- | --- | --- | --- | --- |
| 권한 없음 | 권한이 없을때 작동 | X | string : “권한이 없습니다” | /api/members/forbidden | get |
| 회원가입 | 입력 받은 정보를 바탕으로 회원가입 |  이메일, 비밀번호, 닉네임, 성별, 생일, 관심사, 프로필이미지, 유효성코드 | 가입 여부(아이디) | /api/members/sign-up | post |
| 로그인 | 입력 받은 이메일, 비밀번호를 통해 로그인 | 아이디, 비밀번호  | 아이디, 이메일, 프로필 이미지, 닉네임, 역할, jwt 토큰 | /api/members/login | post |
| 비밀번호 체크 | 입력 받은 비밀번호가 일치하는지 확인 | 아이디, 비밀번호 | 일치 여부(true, false) | /api/members/verify | post |
| 회원 자기소개 수정 | 자기 소개글을 수정한다 | 아이디, 소개글, jwt 토큰 | 수정 여부(아이디) | /api/members/introduce | post |
| 회원 탈퇴 | 회원 탈퇴 | 아이디 | 탈퇴 여부(아이디) | /api/members/{id} | delete |
| 회원 정보 조회 | 회원 정보를 조회 | 아이디 | 아이디, 이메일, 닉네임, 성별, 생일, 역할, 가입일, 소개글, 관심사, 티어, 업적, 쓴리뷰, 팔로잉, 팔로워, 프로필이미지, 오늘 방문 횟수, 총 방문 횟수, 댓글 수 | /api/members/{id} | get |
| 받은 알림 조회 | 회원이 받은 알림 조회 | 아이디 | 아이디, 닉네임, 프로필 이미지, 역할, 알림, 알림 수 | /api/members/{id}/info | get |
| 팔로잉 조회 | 회원의 팔로잉 조회 | 아이디 | 아이디, 닉네임 | /api/members/{id}/following | get |
| 팔로우 조회 | 회원의 팔로우 조회 | 아이디 | 아이디, 닉네임 | /api/members/{id}/follower | get |
| 회원 장르별 티어 조회 | 회원의 장르별 티어, 경험치를 조회 | 아이디, 장르 | 아이디, 재밌어요 티어, 재밌어요 경험치, 재밌어요 퍼센트, 유용해요 티어, 유용해요 경험치, 유용해요 퍼센트, 장르 | /api/members/{id}/genre/{genreNumber} | put |
| 관심사 수정 | 회원의 관심사들을 수정 | 아이디, 관심사 | 수정 여부(아이디) | /api/members/interests | put |
| 닉네임 수정 | 회원의 닉네임 수정 | 아이디, 닉네임 | 수정 여부(아이디) | /api/members/nickname | put |
| 프로필 사진 수정 | 회원의 프로필 사진 수정 | 아이디, 이미지 | string : “프로필 사진 업데이트 성공” | /api/members/profile-image | put |
| 팔로우 등록 | 회원 팔로우 등록 | 본인 아이디, 상대 아이디 | 본인 아이디, 상대 아이디, 상태 | /api/members/follow | post |
| 업적 획득 | 업적 획득 | 아이디, 장르, 업적 | 업적 수 | /api/members/count | put |
| 인증용 이메일 발송 | 인증에 필요한 이메일을 발송 | 이메일, 인증 코드 | string : “이메일이 발송되었습니다.” | /api/email/send | post |
| 이메일 인증 | 이메일 인증 진행 | 이메일, 인증 코드 | string : "잘못된 코드를 입력하였습니다." or "이메일 인증이 완료되었습니다." | /api/email/check | post |
